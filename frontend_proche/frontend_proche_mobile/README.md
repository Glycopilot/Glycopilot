# GlycoPilot - Mobile Proche

Application React Native / Expo dédiée au profil `proche`.

Elle reprend les fonctionnalités de la version web :

- surveillance glycémique du patient ;
- code couleur médical ;
- courbe du jour ;
- événements récents ;
- messagerie Proche - Patient ;
- ajout de pièces jointes ;
- modification et suppression des messages envoyés par le proche.

## Lancer

```bash
cd frontend_proche/frontend_proche_mobile
npm install
npm start
```

Puis ouvrir avec Expo Go ou un simulateur.

Les données sont simulées localement pour valider le parcours avant branchement backend.
